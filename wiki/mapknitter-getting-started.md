# MapKnitter Getting Started

Begin on the MapKnitter.org homepage. Optionally log in to MapKnitter. Choose a descriptive title for the map and let MapKnitter know where on Earth your project is. Select a license for the project data, default is public domain. Choose the base map provider and begin! The project will be saved after the first image is placed.
