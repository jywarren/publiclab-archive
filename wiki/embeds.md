# Embeds

To embed an activity grid from PublicLab.org on another site, use this HTML code (but replace the word `spectrometry` with the tag you'd like):

`<iframe src="http://publiclab.org/embed/grid/spectrometry?t=activities:" style="border:0"></iframe>`

It will appear like this, but within whatever page you place it in:

[![Screen_Shot_2017-05-09_at_12.16.49_PM.png](https://publiclab.org/system/images/photos/000/020/399/large/Screen_Shot_2017-05-09_at_12.16.49_PM.png)](https://publiclab.org/system/images/photos/000/020/399/original/Screen_Shot_2017-05-09_at_12.16.49_PM.png)

### Q&A

For a Questions feature, you can use this:

`<iframe src="http://publiclab.org/embed/grid/infragram?t=questions:" style="border:0"></iframe>`


### Maps

To embed a map of notes from PublicLab.org, use:

`<iframe src="http://publiclab.org/embed/grid/0?t=map:content:40.7:-73.9" style="border:0"></iframe>`
