# Drager Tubes use info

We're using Drager Tubes (diffusion- break the tip for use) as one of the calibration methods for the H2S photostrip test, links to handbook and MSDS sheet on tubes.

Link to handbook on how to use: http://www.draeger.com/media/10/01/87/10018750/tubeshandbook_br_9092086_en.pdf

MSDS sheet: http://www.instrumentdepot.com/images/datasheet/DragerMsds.pdf
